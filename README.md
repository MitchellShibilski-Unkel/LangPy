# LangPy
**Natural Language Processing Model** (SLM) <br />
v1.0 <br />
`pip install git+https://github.com/MitchellShibilski-Unkel/LangPy.git@main`

## Languages Supported
- English | `en`
- Polski *(Polish)* | `pol`
- French | `fr`
- German | `ger`
- Spanish | `esp`
- Italian | `it`

## Features
- Translator
- POS Getter
